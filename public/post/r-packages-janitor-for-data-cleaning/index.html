janitor has simple functions for examining and cleaning dirty data. It was built with beginning and intermediate R users in mind and is optimized for user-friendliness. Import data

For our case-study we are using data from the Tidy Tuesday Project.

# Import Data ----
tuesdata <- tidytuesdayR::tt_load("2018-05-07") 
coffee_chains <- tuesdata$week6_coffee_chains
coffee_chains %>%
  clean_names() %>%
  select(brand, store_number, store_name, country, city) %>%
  get_dupes()
